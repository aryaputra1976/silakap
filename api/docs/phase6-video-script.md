# SILAKAP Phase 6 - Video Training Script

Gunakan script ini untuk rekaman layar singkat. Durasi ideal setiap video 5-10 menit.

## Video 1 - Pengguna OPD

1. Buka halaman login.
2. Login sebagai Pengelola OPD.
3. Tunjukkan menu role OPD.
4. Buat usulan baru.
5. Pilih ASN dan jenis layanan.
6. Upload dokumen.
7. Submit usulan.
8. Buka Pengajuan Aktif.
9. Buka Notifikasi.
10. Buka Download Hasil untuk usulan selesai.

Pesan kunci:

- Pastikan dokumen wajib lengkap.
- Perbaiki usulan dari menu Dikembalikan.
- Download hasil hanya muncul setelah layanan selesai dan output tersedia.

## Video 2 - Analis dan Kabid

1. Login sebagai Analis.
2. Buka antrian verifikasi.
3. Approve satu usulan.
4. Kembalikan satu usulan dengan catatan.
5. Login sebagai Kabid.
6. Buka Approval.
7. Approve usulan.
8. Buka Analytics Kabid.
9. Jelaskan SLA trend, throughput, bottleneck, ranking OPD.

Pesan kunci:

- Catatan pengembalian harus spesifik.
- Dashboard membantu prioritas kerja dan pengendalian SLA.

## Video 3 - Admin Sistem

1. Login sebagai Admin Sistem.
2. Buka Health Dashboard.
3. Jelaskan DB status, storage, backup, dan ENV audit.
4. Jalankan backup manual.
5. Jalankan scan orphan file.
6. Buka Integrasi SIASN.
7. Import Excel sample.
8. Unduh Excel error jika ada.
9. Buka pengaturan email dan kirim test.
10. Buka pengaturan notifikasi multi-channel.

Pesan kunci:

- Backup dan health check wajib sebelum go-live.
- Cleanup file harus diawali dry-run.
- Import SIASN selalu dipantau lewat log.
